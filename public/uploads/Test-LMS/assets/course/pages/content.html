The Assessment is an extended version of the Multiple-Choice ILE. An introduction to the quiz is presented. Feedback for user answers are stored until the assessment has been completed. When completed, the assessment will display a score and a summary of the users responses. A passing score will be recorded. See globals.assessmentUserScore and globals.assessmentRequiredScore.

There should only be 1 assessment per module.
